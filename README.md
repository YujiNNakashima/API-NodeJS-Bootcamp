 # Bootcamp API 

 > Backend API for Bootcamp Application
  
  ##  Usage
  Rename config.env.env to config.env and update values with your own keys.

  ## Install dependencies

  ```
    npm install
  ```
  ## Run App

  ```
  # Run in dev mode
   npm run dev
  
  # Run in prod mode
   npm start
  ```
